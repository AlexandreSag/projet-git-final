# Code of Conduct

## Regles

- Respecter les autres
- Rester poli dans les discussions
- Faire des retours clairs et utiles
- Respecter le travail deja present sur le depot
- Garder des echanges utiles dans les issues et les pull requests

## Signalement

En cas de probleme, ouvrir une issue ou contacter un membre du projet.

## Objectif

Le but est de garder un espace de travail simple, clair et respectueux pour tous les contributeurs.
